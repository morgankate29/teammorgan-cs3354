PREVIEW INFORMATION & HOW IT WORKS
Upon compiling and running the program, the chess game opens in a new window. 
You will see a 8x8 chessboard, with pieces arranged accordingly. 

Starting with white, players will take turns clicking a piece to select, followed by 
clicking the square the player wishes to move the piece to. If the move is invalid, 
the selected piece will simply remain in its original position, and the player will 
have the opportunity to move their piece correctly before switching turns. 

The board updates after each move, and the game ends when a king is captured. A new 
window will confirm who wins, and the program will automatically terminate when the 
game is over. 

In the lefthand corner, the user has the option to start a new game, save the current 
game, and load that existing game under "File". The user can also customize the board's
colors, piece colors, and board size under "Settings" as well.
